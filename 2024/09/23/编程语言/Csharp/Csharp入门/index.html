    知识点一 位运算符

    将数值转换为2进制，在进行位运算

    一、位与 &

    对位运算，有0则0
    int a = 1; //001
    int b = 5; //101
    int c = a & b; //001 所以 c = 1

    二、位或 |

    对位运算，有1则1
    int a = 1; //001
    int b = 1; //011
    int c = a | b; //011 所以c = 3

    三、异或 ^

    对位运算，相同为0，不同为1

    四、位取反

    对位运算，0变1，1变0

    五、左移<<和右移动>>

    左移几位，右侧加几个0
    右移几位，右侧去几个数

    知识点七 变量

    一、变量的范围

    有符号整形变量

    • sbyte -128~128
    • int -21亿~21亿多
    • short -32768~32767
    • long -9百万兆~9百万兆
      无符号整形变量
    • byte 0~255
    • uint 0~22亿多
    • ushort 0~65535
    • ulong 0~18百万兆
      浮点数
    • float 7或8位有效数字
    • double 15~17位有效数字
    • decimal 27~28位有效数字

    二、变量的存储空间

    1 byte = 8bit 1KB = 1024byte 1MB = 1024Kb 1GB = 1024MB 1TB = 1024GB
    存储空间 单位(byte)

    • int uint 4
    • short ushort 2
    • long ulong 8
    • float 4
    • double 8
    • decimal 16
    • bool 1
    • char 2
      sizeof(变量类型) 可以查找变量类型的存储空间
