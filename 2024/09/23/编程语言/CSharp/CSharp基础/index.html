    知识点一 复杂数据类型

    枚举

    一、数组

    1.基本概念

    数组是存储一组相同类型数据的集合
    数组分为 一维、多维、交错数组
    一般情况 一维数组 简称为 数组

    2.数组的申明

    方式一

    变量类型[ ] 数组名;     ——> 只是申明了一个数组 但是并没有开放
    变量类型可以是所有变量类型
    例   

    int[] arr1;

    方式二

    变量类型[ ] 数组名 = new 变量类型[数组的长度];
    例   

    int[ ] arr2 = new int[5];
//相当于开了5个房间，每个房间的值为默认值0

    方式三

    变量类型 [ ] 数组名 = new 变量类型[数组的长度]{内容一, 内容二, …内容五};

    int [ ] arr3 = new int[5]{ 1, 2, 3, 4, 5 };  
//后面填写的内容数量必须与数组长度符合
//开了5个房间，每个房间的值为1、2、3、4、5。

    方式四

    变量类型 [ ] 数组名 = new 变量类型 [ ] {内容1, 内容2, 内容3, …};

    int [ ] arr4 = new int[ ]{1, 2, 3};  
//后面的内容决定了数组的长度

    方式五

    变量类型 [ ] 数组名 = {内容1, 内容2, 内容3,…};

    int [ ] arr5 = {1, 2, 3,...};  不能填多种变量类型

    3.数组的使用

    数组的长度

    数组名.Length   可打印数组的长度

    Console.WriteLine(array.Length);

    获取数组中的元素

    数组中的下标和索引 从0开始的
    通过 索引下标 获取数组中的元素时 要注意不能越界 范围0~Length-1

    Console.WriteLine(array[0]);   //第一个元素
Console.WriteLine(array[1]);   //第二个元素

    修改数组中的元素

    直接赋值

    array[0] = 99;

    遍历数组

    通过循环 快速获取数组中的每一个元素

    for (int i = 0; i < array.Length; i++)
{
    Console.WriteLine(array[i]);
}

    增加数组的元素（搬家）

    数组初始化之后 不能直接添加 元素

    int[] array2 = new int[6]; 
for(int i = 0; i < array.Length; i++)

{
    array2[i] = array[i];                //将旧元素搬到新数组
}
array = array2;                //将原数组改为新数组

    删除数组元素（与增加差不多）

    int[] array3 = new int [5];
for(int i = 0; i < array3.Length; i++)
{
    array3[i] = array[i];
}
array = array3;

    查找数组中的元素（查找数组中某个元素的位置）

    只有通过遍历才能确定数组中是否存储一个目标元素

    int a = 3;
for(int i = 0; i < array.Length; i++)
{
    if( a == array[i])
    {
        Console.WriteLine("和{0}相等的元素在{1}位置", a, i);
        break;
    }
}

    二、二维数组

    1.基本概念

    二维数组 是使用 两个下标（索引）来确定元素的数组
    两个下标 可以理解为 行标 和 列标
    比如矩阵

    1 2 3

    4 5 6

    可以用二维数组 int[2, 3]来表示
    好比两行 三列的数据集合

    2.二维数组的声明

    方式一

    变量类型[,] 二维数组变量名;
    例如         

    int[,] arr;        //声明后要在后面进行初始化

    方式二

    //变量类型[,] 二维数组变量名 = new 变量类型[行,列];
int[,] arr2 = new int[3, 3];

    方式三

    //变量类型[,] 二维数组变量名 = new 变量类型[行,列]{{0行内容1,0行内容2，0行内容3} , {1行内容...}...}
int[,] arr3 = new int[3, 3]{{1, 2, 3}, {4, 5, 6}, {7, 8, 9}};

int[,] arr3 = new int[3, 3]{{1, 2, 3},

                            {4, 5, 6},

                            {7, 8, 9}};

    方式四

    //变量类型[,] 二维数组变量名 = new 变量类型[,]{{0行内容1,0行内容2，0行内容3} , {1行内容...}...}
int[,] arr3 = new int[,]{{1, 2, 3},

                        {4, 5, 6},

                        {7, 8, 9}};

    方式五

    //变量类型[,] 二维数组变量名 = {{0行内容1,0行内容2，0行内容3} , {1行内容...}...}
int[,] arr3 = {{1, 2, 3},

                {4, 5, 6},
                {7, 8, 9}};

    3.二维数组的使用

    二维数组的长度

    Console.WriteLine(array.GetLength(0));        //得到多少行
Console.WriteLine(array.GetLength(1));        //得到多少列

    获取二维数组的元素

    注意：第一个元素索引是0 最后一个元素索引是 长度-1

    Console.WriteLine(array[0,0]);
Console.WriteLine(array[1,2]);

    修改二维数组中的元素

    array[0, 0] = 99;

    遍历二维数组

    for(int i = 0; i < array.GetLength(0); i++)
{
    for(int j = 0; j < array.GetLength(1); j++）
    {
        Console.WriteLine(array[i, j]);
    }       
}

    增加数组的元素

    数组 声明初始化之后 就不能在原有基础上 添加或删除

    int[,] array2 = new int[3, 3];
for(int i = 0; i < array.GetLength(0); i++)
{
    for(int j = 0; j < array.GetLength(1); j++)
    {
        array2[i, j] = array[i, j];         
    }
}
array = array2;
array[2, 0] = 7;
array[2, 1] = 8;
array[2, 2] = 9;

    删除数组中的元素

    int[,] array3 = new int[2, 3];
for(int i = 0; i < array3.GetLength(0); i++)
{
    for(int j = 0; j < array3.GetLength(1); j++)
    {
        array3[i, j] = array[i, j];         
    }
}
array = array3;

    查找数组的元素

    如果在数组中查找元素是否等于某个值 通过遍历形式查找

    知识点二 值类型和引用类型

    一、特殊引用类型string

    1.复习值和引用类型

    值类型——它变我不变，存储在栈内存中，无符号整形、有符号整形、char、bool、结构体
    引用类型——它变我也变，存储在堆内存中，数组（一维、二维、交错）、string、类

    2.string的它变我不变

    string str1 = "123";
string str2 = str1;
//因为string是引用类型，但具备值类型的特征。
//string赋值时，会在堆中重新分配一个空间
//string虽然方便，但有缺点，频繁改变string重新赋值，会产生很多内存垃圾
//有优化替代方案

    知识点三 函数

    一、函数基础

    1.基本概念

    函数（方法）
    本质是一块具有名称的代码块
    可以使用函数（方法）的名称来执行该代码块
    函数（方法）是封装代码进行重复使用的一种机制
    函数（方法）的主要作用:

    • a.封装代码
    • b.提升代码的复用率
    • c.抽象行为

    2.函数写在哪里

    class语句块中
    struct语句块中

    3.基本语法

    static 返回类型 函数名 (参数类型 参数名1, 参数类型 参数名2, …..)
    {
    函数的代码逻辑（可写n多行）
    return 返回值;（如果有返回类型才返回）
    }
    函数名(参数);     使用时写这个即可   

    1. static：不是必须的，在未学习类和结构体前 都必须写
    2. 返回类型：可以写任意变量类型（14种变量类型 + 复杂数据类型），void即是无返回值
    3. 函数名：取名，需使用帕斯卡命名法（MyName），驼峰命名法（myName）
    4. 参数：不是必须的，可有0~n个，类型也可以是任意变量类型（14种变量类型 + 复杂数据类型）
    5. 参数名命名规范按驼峰命名法（myName）。多个参数时，需要使用逗号隔开
    6. return：当返回值类型不为void时，必需通过return返回对应类型的值。（注意：即使是void，也可以选择性使用return）

    4.实际运用

    1. 无参数无返回值的函数
    static void HelloWorld()
{
    Console.Write("你好，世界");
    return;        //可省略
}
    1. 有参无返回值的函数
    static void SayYourName(string name)
{
    Console.Write("你的名字是{0}", name);
}

//使用时:
SayYourName("聪哥");        //括号里可以填变量名，一个string常量，甚至是返回值为string类型的函数
    1. 无参数有返回值函数
    static string WhatYourName()
{
    return "聪哥";       
}
//使用时:
SayYourName(WhatYourName());
//有返回值的函数，拿函数的返回值用或拿变量接受它的结果，也可以直接调用但返回值没用了。
    1. 有参数有返回值的函数
    static int Sum(int a, int b,)
{
    return a + b;       //return后面可写表达式
}
//使用时：
Console.Write(Sum(2, 5));
    1. 有参数有多返回值的函数
    static int[] Calc(int a, int b)
{
    int sum = a + b;
    int avg = sum / 2;
    int[] arr = { sum, avg };
    return arr;        //或者return new int[] { sum, avg };
}
//使用时：
int[] arr = Calc(5, 7);

    5.关于return

    即使函数(void)没有返回值，也可以使用return
    return可以直接不执行之后的代码，直接返回到函数外部
    即return之后的代码都不会执行

    二、ref和out知识点

    1.学习ref和out的原因

    可以解决 在函数内部改变外部传入的内容 里面变了 外面也要变

    2.使用

    函数参数的修饰符
    当传入的值类型参数在内部修改时 或引用类型参数在内部重新申明时
    外部的值就会发生改变

    1. ref
    a.变量
static void ChangRef(ref int value)
{
    value = 3;
}
使用时：
int a = 1;
Chang(ref a);        //注意使用时也要在变量前加ref
Console.Write(a);        //这时候a的值就可以变为3，若没有ref修饰不能变化
b.数组
static void ChangeValue (ref int[] arr)
{
    int[] arr = {1, 2, 3};
}
使用时：
int [] arr2 = {55, 3, 6};
ChangeValue(ref arr2);
Console.Write(arr2[0]);        //值会发生改变，变为1
    1. out例子
    static void ChangOut(out int value)
{
    value = 3;
}
使用时：
int a = 1;
Chang(out a);        //注意使用时也要在变量前加out
Console.Write(a);        //这时候a的值就可以变为3，若没有out修饰不能变化

    3.ref和out的区别

    • ref传入的变量，必须初始化，out不用
    • out传入的变量，必须在内部赋值，ref不用

    三、变长参数和参数默认值

    2.变长参数关键词

    举例        利用函数计算n个整数的和

    //从前：
static int Sum(int a, int b, int c, .....)
{    return 0;    }
//使用变长参数：params
static int Sum(params int[] arr)        
{
    int sum = 0;
    for(int i = 0; i < arr.Length; i++)
    {
        sum += arr[i];
    }        
    return sum;
}        

    注意：

    • 所有传入的参数都会计入这个数组内
    • arams关键字后面必为数组
    • 数组的类型可以是任意类型
    • 函数参数可以有别的参数和params关键字修饰的参数
    • 函数参数中最多只能出现一个params关键字，并且一定是最后一组参数，前面可以有n个其他参数

    3.参数默认值

    有参数默认值的参数可称为 可选参数
    作用：当调用参数时，可以不传入参数，不传入参数时，就会使用默认值作为参数的值
    例如

    static string Speak(string str= "无话可说")       
//这个"无话可说"就是参数默认值，若传入参数则参数值为传入的值

    注意：

    • 支持多参数默认值，每个参数都可以有默认值。
    • 若要混用，可选参数必须写在普通参数后面。

    四、函数重载

    1.基本概念

    在同一语句块中（class或struct）中
    函数（方法）名相同
    参数的数量不同 或 参数的数量相同，但参数的类型或顺序不同
    作用：

    • 命名一组功能相似的函数，减少函数名的数量，避免命名空间污染
    • 提升程序可读性

    2.实例

    注意：

    • 重载和返回值类型无关，只和参数类型、个数、顺序有关
    • 调用时，程序会自己根据传入的参数类型判断使用哪一个重载
    static int CalcSum(int a, int b)
{
    return a + b;
}
    1. 参数数量不同
    static int CalcSum(int a, int b, int c, int d)
{
    return a + b + c + d;
}
    1. 数量相同，类型不同
    static float CalcSum(float a, float b)
{
    return a + b;
}
static float CalcSum(float a, int b)
{
    return a + b;
}
    1. 数量相同，顺序不同
    static float CalcSum(int a, float b)
{     
    return a + b;
}
    1. ref 和 out
    //ref和out 可理解为是一种变量类型，但ref和out不能同时修饰
static float CalcSum(ref int a, float b)        //加入ref或out修饰，可理解为是另一种变量
{
    return a + b;
}
    1. 变长参数
    static float CalcSum(int a, float b, params int[] arr)
{     
    return a + b;
}

    五、递归函数

    1.基本概念

    让函数自己调用自己
    条件：

    • 必须有结束调用的条件
    • 用于条件判断的条件必须能达到停止的目的

    2.实例

    用递归函数打印0到10

    static void Fun(int a)
{
    if(a > 10)
    {
        return;
    }
    Console.WriteLine(a);
    ++a;
    Fun(a);
}
//使用时：
Fun(0);

    知识点四 结构体

    1.基本概念

    是一种自定义变量类型 类似枚举需要自己定义
    是数据和函数的集合
    在结构体中 可以申明各种变量和方法
    作用：用来表现存在关系的数据集合，比如表现学生、动物等

    2.基本语法

    结构体一般写在namespace语句块中
    结构体关键字：struct
    struct 自定义结构体名字
    {      
    1.变量
    2.构造部分（可选）
    3.函数
    }
    注意：结构体名字规范是帕斯卡命名法

    3.实例

    struct Student
{
    //变量        在结构体中声明的变量不能直接初始化        变量类型可以写任意类型，包括结构体，但不能结构体本身。
    public int age;
    public bool sex;
    public int number;
    public string name;
    //函数方法        表现这个数据结构的行为        可写无数个函数
    //注意：在结构体中的方法，目前不需要加static关键字
    public void Speak()
    {
        //在函数中可以直接使用结构体内部声明的函数。
        Console.WriteLine("我的名字是{0}，我今年{1}岁了", name, age);
    }
    //构造函数
    public Student (int age, bool sex, int number, string name)
    {

        //使用this关键字        构造体中的变量 = 外部传入的变量
        this.age = age;
        this.sex = sex;
        this.number = number;
        this.name = name;
    }
}

    4.结构体的使用

    在Main函数中使用

    //变量类型 变量名
Student s1;
s1.age = 10;
s1.sex = false;
s1.number = 110;
s1.name = "马里奥";
s1.Speak();

    5.访问修饰符

    修饰结构体中的变量和方法，是否能被外部使用

    • publie    可在外部使用
    • private      只能在内部使用
      默认不写为private

    6.结构体的构造函数(一般用于在外部方便初始化)

    基本概念：

    • 没有返回值
    • 函数名必须和结构体名相同
    • 必须有参数
    • 如果申明了构造函数，那么必须在其中对所有变量数据初始化
    • 申明在结构体中
    //使用时：
Student S2 = new Student(18, true, 2, "路易基");
