    知识点一 MVC的基本概念

    一、MVC的历史

    是一种软件开发的编程思想
    自定前提：粗略地将计算机上的使用工具分为应用软件、网页、游戏
    MVC是应用软件开发、网页开发最常用和流行的通用开发框架
    应用软件和网页基本是由UI、Data(数据)构成

    二、MVC的基本概念

    全名是 Model View Controller
    模型(Model) - 视图(View) - 控制器(Controller) 的缩写
    是一种软件设计规范，用一种业务逻辑、数据、界面显示 分离的方法组织代码
    将业务逻辑聚集到一个部件里，再改进和个性化定制界面及用户交互的同时，不需要重新编写业务逻辑

    三、MVC在游戏中的应用

    注意：
    它不是必备内容
    它主要用于开发游戏UI系统逻辑

    将UI面板逻辑一分为三，数据、界面、业务逻辑

    好处：降低耦合，方便修改、逻辑清晰，适合协同开发
    坏处：脚本变多，体量变大，流程变复杂
