知识点一 导入Excel相关DLL包

一、了解Excel表的本质

Excel表本质就是数据，有自己的存储读取规则
如果我们想要通过代码读取它
那么必须知道它的存储规则

官网提供了对应的DLL文件用来解析Excel文件

DLL文件：库文件，可以理解为许多代码的集合
将相关代码集合在库文件中，可以方便迁移和使用
有了某个DLL文件，我们就可以使用其中已经写好的代码

而Excel的DLL包就是官方已经把解析Excel表的相关类和方法写好了
方便用户使用

二、导入官方提供的Excel相关DLL文件

直接将 .dll 后缀的文件导入到工程文件夹下

知识点二 Excel数据读取

一、打开Excel表

知识点

  1. FileStream读取文件流
  2. IExcelDataReader类，从流中读取Excel数据
  3. DataSet 数据集合类 将Excel数据转存进其中方便读取

代码

using Excel;

using(FileStream fs = File.Open(application.dataPath + "PlayerInfo.xlsx", FileMode.Open, FileAccess.Read))
{
    //通过我们的文件流获取Excel数据
    IExcelDataReader excelReader = ExcelReaderFactory.CreateOpenXmlReader(fs);
    //将Excel表中的数据转换为DataSet素具类型，方便我们获取其中的内容
    DataSet dataSet = excelReader.AsDataSet();
    //得到Excel文件中的所有表信息，遍历所有表
    for(int i = 0; i < dataSet.Tables.Count; i++)
    {
        //打印表的名字
        print( dataSet.Tables[i].TableName );
        //打印表的行数
        print( dataSet.Tables[i].Rows.Count );
        //打印表的列数
        print( dataSet.Tables[i].Columns.Count );
    }
    fs.Dispose();
}

二、获取Excel表中单元格的信息

知识点

  1. FileStream读取文件流
  2. IExcelDataReader类，从流中读取Excel数据
  3. DataSet 数据集合类 将Excel数据转存进其中方便读取
  4. DataTable 数据表类 表示Excel文件中的一个表
  5. DataRow 数据行类 表示某张表中的一行数据

代码

using( FileStream fs = File.Open(application.dataPath + "PlayerInfo.xlsx", FileMode.Open, FileAccess.Read))
{
    //通过我们的文件流获取Excel数据
    IExcelDataReader excelReader = ExcelReaderFactory.CreateOpenXmlReader(fs);
    //将Excel表中的数据转换为DataSet素具类型，方便我们获取其中的内容
    DataSet dataSet = excelReader.AsDataSet(); 
    //得到其中第一张表的数据
    DataTable table = dataSet.Tables[0];
    //得到表中第一行的数据
    DataRow dataRow = table.Row[0];
    //得到该行的第一列的数据，返回值是一个object类型
    object obj = dataRow[0];
}

三、获取Excel表中信息对于我们的意义

既然能获取Excel表中的所有数据
那么我们就可以根据表中数据来动态的生成相关数据

  1. 数据结构类
  2. 容器类
  3. 2进制数据

为什么不直接读取Excel表而要将他转成2进制数据？

  1. 提升读取效率
  2. 提升数据安全性
